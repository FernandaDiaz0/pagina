PRODUCTOS

¿Que tanto se necesita?

La cantidad de ingredientes y utensilios necesarios para la repostería dependerá del tipo específico de producto que estés preparando.

UTENCILIOS BASICOS

  • Batidora:
    Para mezclar y batir ingredientes.
  • Recipientes para mezclar:
    Tazones de diferentes tamaños para combinar ingredientes.
  • Moldes para hornear:
    Varían según el producto (pasteles, galletas, muffins, etc.).
  • Rodillo:
    Utilizado para estirar la masa en recetas de pasteles o galletas.
  • Espátulas y cucharas de medir:
    Para medir y transferir ingredientes.
  • Termómetro de horno:
    Para asegurarse de que el horno esté a la temperatura correcta.
  • Cortadores de galletas y moldes para decoración:
    Opcional, pero útil para productos más decorativos.



INGREDIENTES BASICOS

  • Extractos y esencias:
    Como vainilla, almendra, etc., para agregar sabor.
  • Frutas, nueces, chocolate:
    Para añadir textura y sabor adicional.
  • Fondant o glaseado:
    Para decorar pasteles y postres.

    • Además, es importante tener en cuenta que la higiene y la precisión en las mediciones son cruciales en repostería. Además de estos elementos, siempre es útil tener una buena comprensión de las técnicas básicas de repostería, como el horneado, el glaseado y la decoración.
    La cantidad exacta de ingredientes y utensilios dependerá de la receta específica que estés siguiendo. Es recomendable leer cuidadosamente la receta antes de empezar y asegurarte de tener todo lo necesario antes de comenzar a hornear.
